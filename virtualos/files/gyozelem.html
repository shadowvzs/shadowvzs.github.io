Gyozelem.ro

- Church presentational site -

Account management, private message sending, audio player, gallery etc.


"this site was started in 2011, it was for desktop os, MPA, iframes.
Site was rewrote twice, last time in 2018 and it was changed alot!"



  • Main features:
    • Structure:
      • - Separated frontend and backend, communication with requests
      • - Single js and component structure, shared data object between components
      • - PHP7 allways return JSON, no direct interaction (replacable with other backend)
      • - Option for input validation in client & php side with notify message
      • - Javascript routing (a lil from .htaccess), model, view (template functions) etc
      • - Single Page Application, page never refreshing during you use it, only update DOM pieces
      • - Responsive design with grid (and mobil first fallback if device not support grid)
    • Basics:
      • - Login, registration, user account administration
      • - User account administration for admin
      • - Message sending to users
      • - Notification message at right-top corner
      • - Audioplayer which allow to play mp3
      • - Message sending to users
      • - Guestbook, calendars (for events, news)
      • - Gallery (paralel file upload with progressbar for admin)
      • - Youtube video listing (from api)
