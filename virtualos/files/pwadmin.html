pwAdmin

- PHP pwAdmin for Perfect World -

Account management, item builder and item webshop for Perfect World game.
send item to game character.


"this site a bit outdated because i did at the end of 2016.
This is for fun project and i don't maintain or sell game client or server files!"



  • Main features:
    • Runtime setup wizard:
      • - progress bar
      • - configure MySQL connection
      • - user password encryption
      • - setup version & rewards
      • - create/update admin user
    • Account management:
      • - login, registration, user account administration
      • - user account administration, user info
      • - filter userlist, reward users, change rank etc
      • - create and visualize item and octets
      • - create/update admin user
    • Server management:
      • - ability to select sender role, channel
      • - send message to game server where will be sent to game clients
    • Item builder:
      • - collected many item type with different octet rules
      • - create, customize, visualize item
      • - real time octet code building
      • - create and visualize item and octets
      • - send item directly to role id with socket in packet form
      • - personalize item crafter name, edit webshop attributes
    • Webshop:
      • - categorized item list with filters
      • - get role list from server which connected with logged account
      • - real visualized stats which based on game formula
      • - ability to buy item with web point and in game gold (for v1.4.2 v27)
