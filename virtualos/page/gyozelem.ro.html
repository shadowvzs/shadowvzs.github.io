gyozelem.ro


- started working on it in 2011, currently only maintaining it

Home page of my church. This is a web page written from scratch without using any framework.

Main features:
  • user registration and login with four hierarchical levels: registered, member, moderator, admin having different rights and access inside the web page
  • internal message sending between two users
  • guest book
  • blog like part with article writing and listing
  • events/news history, next most recent event/new shown on the home page
  • photo gallery: it can only be managed by the admin user (album creation, photo description, photo moving to other album, single or batch delete of photos, full screen left right navigation, photo navigation by clicking the thumbnails from the bottom bar)
  • online bible in three languages with features written from scratch:
    • search
    • navigation
    • parallel reading between two versions or current version and footnotes - hu
    • settings: options for setting the color, font, size, default bible version, spacing between bible verses
    • (it is a standalone web page hosted on other domain but linked from gyozelem.ro)
    • custom dialog windows, menu system used all over the web page

Technologies: php, mysql, html, css, javascript
